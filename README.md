## Local development
Define a `.env` file within the root folder and define the following variables e.g:
```
REACT_APP__SERVER_ORIGIN=http://localhost:3001
REACT_SERVER__FACEBOOK_APP_ID=xxxx
REACT_SERVER__FACEBOOK_APP_SECRET=xxxx
```

Start the server
```
yarn start
```
